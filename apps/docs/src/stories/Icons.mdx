# Icon System

The Prismia Design System uses **lucide-react** as the official icon library.

## Installation

```bash
npm install lucide-react
```

## Usage

Import icons directly from lucide-react:

```jsx
import { Home, User, Search } from 'lucide-react';

function MyComponent() {
  return (
    <>
      <Home size={24} />
      <User size={24} color="#3C3F45" />
      <Search size={20} strokeWidth={1.5} />
    </>
  );
}
```

## Icon Props

All lucide-react icons accept the following props:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | number | 24 | Icon size in pixels |
| `color` | string | currentColor | Icon color |
| `strokeWidth` | number | 2 | Stroke width |
| `className` | string | - | CSS class name |

## Best Practices

- Use `size={24}` for standard UI icons
- Use `size={20}` for inline text icons  
- Use `strokeWidth={1.5}` for a lighter appearance
- Always provide semantic colors from design tokens
- Use `aria-label` or `aria-hidden` for accessibility
